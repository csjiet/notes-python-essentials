
# matplotlib.pyplot = `plt`
## Axis units
> plt.xlim(); plt.ylim()
```python
import matplotlib.pyplot as plt
# y-axis
plt.ylim(-1000, 60000) # Limits the plot of y axis at -1000 to +60,000

# x-axis
plt.xlim(0, 60000) # Limits the plot of x axis at 0 to +60,000
```


## Subplots
- Subplots are called "sub"-"plots" because they help define group of smaller axes that can exist together within a single figure. E.g., This helps to compare different views side by side.
```python
import matplotlib.pyplot as plt
# Creates row*col "plots" (arranged accordingly in a matrix fashion), each plot with size width*height, in a single figure.
fig, ax = plt.subplots({row}, {col}, figsize=(width, height))

'''
Args:

Return:
- fig (<class 'matplotlib.figure.Figure'>) - Figure static class 
- ax (<class 'numpy.ndarray'>) - an array of each row*col plot's axis
'''
```

- **Plot hierarchy (upon initialization is assigned to a variable)** 
-----
	- ("`plt`") matplotlib.pyplot/ plot  - `import matplotlib.pyplot as plt`
		- ("`fig`") matplotlib.pyplot.figure  - `fig = plt.figure()`
		-  ("`ax`") matplotlib.pyplot.axes  - `ax = plt.axes()`

-----
   
- **Plot hierarchy with callable function**
	- ("`import matplotlib`")**matplotlib***
		- ("`plt`") **matplotlib.pyplot**- `import matplotlib.pyplot as plt`
			- (***Normal function*** defined within - "matplotlib.pyplot")
				- N> `matplotlib.pyplot.figure()` 
				- N> `matplotlib.pyplot.axes()` 
					- N> `matplotlib.pyplot.axes().view_init(elev, azim)` - [[elevation and azimuth as viewpoints of 3d plot]]
				- N> `matplotlib.pyplot.subplots()`
				- N> `matplotlib.pyplot.subplot()`
				- N> `matplotlib.pyplot.subplot_adjust()`
			- ("`fig`") **matplotlib.pyplot.figure**  - `fig = plt.figure()`
				- (***Utility function*** - "matplotlib.figure" (not the same as "matplotlib.pyplot.figure", are callable from within matplotlib.pyplot)
					- U> `matplotlib.figure.add_subplot()` 
					- U> `matplotlib.figure.add_subfigure()` 
					- U> `matplotlib.figure.add_gridspec()` 
					- U> `matplotlib.figure.add_callback()` 
					- U> `matplotlib.figure.add_axobserver()` 
					- U> `matplotlib.figure.add_artist()` 
					- U> `matplotlib.figure.add_axes()` 
			-  ("`ax`") **matplotlib.pyplot.axes**  - `ax = plt.axes()`
				- (***Utility function*** - "matplotlib.axes" (not the same as "matplotlib.pyplot.axes", are callable from within matplotlib.pyplot)
					- U> `matplotlib.axes.Axes.plot()` 
> Note that there are utility functions WITHIN pyplot vs utility functions with the same name (but belongs to matplotlib's utility function)
