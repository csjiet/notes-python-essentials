
# List
`<type 'list'>` + `<type 'list'>` = `<type 'list'`

``` python
arr1 = [1,2,3] 
arr2 = [4, 5]

# Arithmetic: class list defined __add__() to extend lists
arr = arr1 + arr2
print(arr)
# Output: [1,2,3,4,5]

# Function class
arr = arr1.extend(arr2)
print(arr)
# Output: [1,2,3,4,5]

```
